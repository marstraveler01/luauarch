# Welcome to Luauarch

**Luauarch** is a keyless, non-streamlined Roblox script hub designed for direct functionality and ease of use.

## Execution

To load the script hub, run the following code in your executor:
```luau
loadstring(game:HttpGet("https://raw.githubusercontent.com/marstraveler01/luauarch/refs/heads/main/loader.luau"))()
```
## Supported Games

Luauarch currently supports the following games:
- An Annoyingly Long Monorail Game
- Zyltex's Animations

## Features

- **Keyless Access**: No external key systems or checkpoints required.
- **Non-Streamlined**: Focused on raw script performance rather than heavy UI overlays.
- **Minimalist**: Optimized for fast loading and low resource usage.
